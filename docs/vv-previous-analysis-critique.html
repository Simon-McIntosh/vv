Critique of the Previous VV Rattle Analysis

2026-05-27 · review of three figures from an earlier analysis (assembly_team_summary.png, feasible_envelope.png, rattle_analysis.png), embedded below and assessed against the current peak-to-peak / n=1 findings. Source figures archived in this document.

Bottom line: the previous analysis got the nominal LP geometry exactly right (envelope ±1.547/±1.523 mm, rotation ±187.5 µrad, nominal range 3.093 mm — all identical to the current results), which is reassuring. But it made one critical statistical error: it asserts the total peak-to-peak rattle range is a constant 3.093 mm "delta function" for every assembly. It is not — the range varies with assembly state (current MC: median 2.24, P95 3.01, min 0.23 mm) and a near-alternating assembly locks the vessel to ≈ 0. The previous analysis also mixes several different "rattle" metrics, producing three inconsistent percentile numbers in the same deck.

  • Maximum rotation: ±0.1875 mrad = ±187.5 µrad — identical, and correctly an independent DOF.

This cross-check is valuable: two independent analyses agree on every nominal number, so the LP geometry is not in doubt.

The critical error ✗ — "the range is constant"

The bottom-left panel of rattle_analysis.png plots the total rattle range as a delta function at 2×LP = 3.093 mm and labels it "CONSTANT". The reasoning was evidently: total range = 2 × (nominal half-bound) = 3.093 mm, the same for every assembly.

Why it is wrong. The "difference bound" A·Δq ∈ [−3, 3]⁹ is indeed independent of the assembly offsets, but attaining it requires two feasible vessel positions whose difference is the full-diameter vector — which is only possible when the offsets leave room. For offset assemblies they do not, so the range is ≤ 3.093 mm and varies. The current 5,000-sample MC measures it directly: median 2.24, P95 3.01, P99 3.09, min 0.23 mm, and the alternating ±1.5 mm pattern locks the vessel to ≈ 0. The previous analysis never computed the per-assembly range — it asserted a constant — so it missed both the variation and the locking phenomenon entirely.

Tellingly, the analysis is internally inconsistent: the top-left panel of the same figure colours assembly states by "max rattle" over a range of 1.6–3.0 mm (clearly varying), while the bottom-left claims the range is a constant 3.093 mm. The same figure shows the rattle both varying and constant.

This is exactly the one-sided-reach vs peak-to-peak-range confusion diagnosed in the current report. The current analysis fixes a single metric — the peak-to-peak range — and uses it consistently (P95 = 3.01 mm, ceiling 3.093 mm), tied to its physical meaning.

Missing: the physical consequence

The previous analysis framed everything as an "assembly tolerance" deliverable. It never connects the lateral rattle to why it matters: that a horizontal shift is an n = 1 first-wall displacement driving start-up heat-load localisation on the inner wall, and that EM-driven motion impacting the end-stops is a structural-dynamics and fatigue concern. The current report makes both consequences explicit.

Net assessment. The previous analysis is a sound geometric precursor — its nominal numbers are exactly right and corroborate the current LP. As a conservative design bound its 3.093 mm even happens to be correct (it is the ceiling). But its central statistical claim (constant range) is wrong, it mischaracterises the typical case (the median is 2.24 mm, not 3.09 mm), it misses that assembly can lock the vessel, and it conflates several rattle metrics. The current peak-to-peak / n=1 analysis supersedes it.
